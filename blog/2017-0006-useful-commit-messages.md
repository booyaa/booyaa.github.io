Keeping a copy of this excellent bit of advice until I've committed (no pun) it
to memory.

```text
feat: add hat wobble
^--^  ^------------^
|     |
|     +-> Summary in present tense.
|
+-------> Type: chore, docs, feat, fix, refactor, style, or test.
```
